## Возможности

- 🎥 **Захват движений** с веб-камеры или видеофайла
- 🧠 **3D-трекинг руки** через MediaPipe World Landmarks (координаты в метрах)
- 📐 **Кинематический анализ** — прямая и обратная задача, проверка диапазонов
- ⚡ **Биомеханический расчёт** моментов: M(θ) = m·g·l·sin(θ)
- 🎮 **MuJoCo-симуляция** двухзвенной руки с визуализацией
- 🔧 **Подбор привода** из базы 25+ моторов и 11 редукторов (РФ + Китай)
- 📊 **Графики и отчёты** — CSV, JSON, PNG

## Быстрый старт

```bash
# 1. Клонировать репозиторий
git clone <repo-url>
cd aux-drive-picker

# 2. Установить зависимости
pip install -r requirements.txt

# 3. Запустить лаунчер
python aux-dp.py
```

Лаунчер предложит меню для настройки всех параметров без командной строки.

## Использование

### Лаунчер (рекомендуется)

```bash
python aux-dp.py
```

Меню:
- **1** — выбрать источник (камера / видео / свой файл)
- **2** — выбрать руку (левая / правая)
- **3** — задать нагрузку (0 / 2 / 5 кг)
- **4** — вкл/выкл MuJoCo
- **5** — вкл/выкл графики
- **7** — 🚀 запустить проект

### Командная строка

```bash
# Базовый запуск (видео по умолчанию)
python main_pipeline.py

# Веб-камера, правая рука, груз 2 кг
python main_pipeline.py --camera 0 --side right --load 2.0

# Только расчёты, без MuJoCo и графиков
python main_pipeline.py --no-mujoco --no-plots

# Своя папка для результатов
python main_pipeline.py --output my_results
```

### Тестирование отдельных этапов

```bash
# Тест компьютерного зрения
python vision/test_vision.py

# Тест кинематики + биомеханики + MuJoCo
python biomechanic/test_biomechanics.py
```

## Этапы проекта

| Этап | Модуль | Описание |
|------|--------|----------|
| **1** | `vision/` | MediaPipe Pose → 3D-координаты, длины звеньев, углы |
| **2** | `kinematics/` | Прямая/обратная кинематика, диапазоны 0°–130°, координаты запястья |
| **3** | `biomechanics/` | M(θ) = m·g·l·sin(θ), Mmax для 0/2/5 кг, MuJoCo-валидация |
| **4** | `actuators/` | База 25 моторов + 11 редукторов (РФ + КНР), оптимизация пары |
| **5** | `main_pipeline.py` | Сквозной прогон: видео → привод |
| **6** | `outputs/` | Валидация на видео, графики, CSV |
